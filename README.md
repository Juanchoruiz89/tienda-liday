# Tienda Liday - E-commerce de Moda

Tienda Liday es una página web de comercio electrónico dedicada a la venta de ropa y accesorios de moda. El proyecto implementa todas las funcionalidades de un e-commerce moderno, incluyendo consumo de API, carrito de compras, formulario de contacto y diseño responsivo.

## Características del Proyecto

### 1. Estructura Semántica HTML

- Uso de etiquetas semánticas: `<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`
- Metaetiquetas optimizadas para SEO
- Atributos `alt` en imágenes para accesibilidad

### 2. Diseño Responsivo con CSS

- Implementación con Bootstrap 5 y CSS personalizado
- Flexbox para la sección de productos
- Grid CSS para la sección de reseñas
- Media queries para adaptación a diferentes dispositivos

### 3. Funcionalidades JavaScript

- Consumo de API REST (Fake Store API) para obtener productos
- Renderizado dinámico de productos en el DOM
- Carrito de compras con funcionalidad completa:
  - Añadir/eliminar productos
  - Actualizar cantidades
  - Cálculo de totales en tiempo real
- Persistencia del carrito usando localStorage
- Validación de formulario de contacto

### 4. Formulario de Contacto

- Validación en tiempo real de campos
- Integración con Formspree para manejo de envíos
- Mensajes de confirmación al usuario

### 5. Accesibilidad y SEO

- Navegación con teclado
- Textos alternativos para imágenes
- Metaetiquetas descriptivas
- Estructura HTML semántica

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos personalizados, Flexbox, Grid
- **Bootstrap 5**: Framework para diseño responsivo
- **JavaScript (ES6+)**: Lógica e interactividad
- **Fake Store API**: Fuente de datos de productos
- **Font Awesome**: Iconos
- **Google Fonts**: Tipografía
